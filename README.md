# SID Portfolio

Premium cinematic portfolio for Siddhartha Abhimanyu.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Deployment

This project is optimized for Vercel. Build command: `npm run build`. Output directory: `dist`.

Update canonical URLs in `index.html`, `public/robots.txt`, and `public/sitemap.xml` after assigning the final production domain.
